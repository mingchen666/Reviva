# 复习策略选择规则

## 一、策略矩阵

| 剩余时间 | 策略名称 | 核心原则 | 每日重点 |
|----------|----------|----------|----------|
| ≥ 21天 | 从容复习 | 全面覆盖，不留死角 | 每章 2-3 天 |
| 14-20天 | 系统复习 | 全面覆盖，重点突出 | 每章 1-2 天 |
| 7-13天 | 重点突破 | 优先高频考点和薄弱项 | 高频考点优先 |
| 3-6天 | 冲刺模式 | 只看核心考点+错题 | 速记+刷题 |
| 1-2天 | 考前速记 | 总纲+公式卡+易错清单 | 背诵为主 |

## 二、策略详情

### 从容复习（≥21天）
- 第 1 轮：通读教材，梳理知识框架（7-10天）
- 第 2 轮：做练习题，巩固知识点（7-10天）
- 第 3 轮：模拟考试，查漏补缺（3-5天）
- 最后 2 天：考前速记

### 系统复习（14-20天）
- 第 1 轮：按章节梳理知识点（7-10天）
- 第 2 轮：专项突破薄弱点（4-5天）
- 第 3 轮：模拟考试（2-3天）
- 最后 1-2 天：考前速记

### 重点突破（7-13天）
- 第 1 天：快速过一遍知识总纲
- 第 2-5 天：高频考点逐个突破 + 练习
- 第 6-8 天：错题整理 + 巩固练习
- 第 9-10 天：模拟考试
- 最后 1-2 天：考前速记

### 冲刺模式（3-6天）
- 每天：速记页 + 公式卡 + 高频考点
- 每天：一套模拟卷或专项练习
- 错题优先，新题为辅
- 最后一天：只看总纲和错题

### 考前速记（1-2天）
- 只看速记页、公式卡、易错清单
- 不做新题
- 考前 2 小时：公式默记 + 高频考点扫一遍
- 考前 30 分钟：深呼吸，调整心态

## 三、触发条件自动判断

```
if 剩余天数 >= 21:
    策略 = "从容复习"
elif 剩余天数 >= 14:
    策略 = "系统复习"
elif 剩余天数 >= 7:
    策略 = "重点突破"
elif 剩余天数 >= 3:
    策略 = "冲刺模式"
else:
    策略 = "考前速记"
```

## 四、用户偏好调整

- 用户说"我要全面复习" → 策略上浮一级（如冲刺→重点突破）
- 用户说"没时间了" → 策略下沉一级
- 用户说"这门课我学得很好" → 减少该科时间，分配给其他科目
