# 时间分配原则

## 一、每日时间分配

### 黄金时间法则
| 时间段 | 精力水平 | 适合任务 |
|--------|----------|----------|
| 09:00-11:00 | ★★★ 最高 | 难度大的科目、新知识学习 |
| 11:00-12:00 | ★★☆ 较高 | 练习题、巩固 |
| 14:00-16:00 | ★★☆ 较高 | 中等难度科目 |
| 16:00-17:00 | ★☆☆ 一般 | 练习题、错题整理 |
| 19:00-21:00 | ★★☆ 较高 | 复习回顾、记忆类内容 |
| 21:00-22:00 | ★☆☆ 一般 | 当日总结、明日计划 |

### 每日结构模板
```
| 时间段 | 任务类型 | 时长 |
|--------|----------|------|
| 上午 | 核心学习（新知识/难点） | 3h |
| 下午 | 练习巩固 | 2-3h |
| 晚上 | 回顾复习 | 1-2h |
| 合计 | | 6-8h |
```

## 二、科目时间权重

### 权重计算公式
```
科目权重 = 考试分值占比 × 难度系数 × 紧急系数
```

| 因素 | 说明 | 取值 |
|------|------|------|
| 考试分值占比 | 该科目在总成绩中的比重 | 0.1-1.0 |
| 难度系数 | 自评掌握程度 | 掌握好=0.7, 一般=1.0, 差=1.5 |
| 紧急系数 | 考试先后顺序 | 最先考=1.5, 最后考=0.7 |

### 时间分配示例（3门课）
| 科目 | 权重 | 每日时间 |
|------|------|----------|
| 高数（难，先考） | 1.5 | 3h |
| 英语（一般，中间考） | 1.0 | 2h |
| 政治（易，后考） | 0.7 | 1.5h |

## 三、休息与节奏

### 番茄钟节奏
- 每学习 45-50 分钟，休息 10 分钟
- 每 2-3 小时，休息 20-30 分钟
- 每天保证 7-8 小时睡眠

### 弹性时间
- 每天预留 1 小时弹性时间
- 用于：处理未完成的任务、应对突发情况
- 如果没有突发情况，用于复习薄弱知识点
