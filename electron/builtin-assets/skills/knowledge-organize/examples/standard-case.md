# 标准案例：有完整资料的知识梳理

## 用户输入

> 帮我整理一下高等数学（上册）的知识点，资料在 `高数讲义.pdf`

## 处理过程

### 1. 读取资料
使用 Read 工具读取 `高数讲义.pdf`，分批读取（每次 20 页）。

### 2. 分析内容
识别出 6 个章节：
- 第一章：函数与极限
- 第二章：导数与微分
- 第三章：中值定理与导数应用
- 第四章：不定积分
- 第五章：定积分
- 第六章：定积分应用

### 3. 选择模板
用户没有指定格式，默认使用 **outline-template.md**（章节大纲）+ **one-page-summary-template.md**（一页纸总纲）。

### 4. 生成输出
先输出一页纸总纲，再按章节输出详细大纲。
