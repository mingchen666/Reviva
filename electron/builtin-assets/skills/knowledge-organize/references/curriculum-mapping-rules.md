# 知识点映射与分类规则

## 一、章节结构识别

### 标题层级判定
- 一级标题（`#`）→ 章 / 大主题
- 二级标题（`##`）→ 节 / 子主题
- 三级标题（`###`）→ 知识点群组
- 正文段落 → 具体知识点

### 无明确标题时的识别策略
1. 通过编号模式识别：`1.`、`1.1`、`第一章`、`Chapter 1`
2. 通过字体/格式识别：加粗、下划线、字号差异
3. 通过内容模式识别：`本章小结`、`学习目标`、`重点`等关键词
4. 通过分隔符识别：横线 `---`、星号 `***` 等

## 二、知识点分类标签

| 标签 | 含义 | 识别特征 |
|------|------|----------|
| `【定义】` | 概念定义 | "是指"、"定义为"、"即"、"所谓" |
| `【公式】` | 数学公式/表达式 | 含 `=`、数学符号、变量 |
| `【定理】` | 定理/定律/原理 | "定理"、"定律"、"原理"、"法则" |
| `【对比】` | 易混淆概念对比 | "与…的区别"、"不同于"、"相比之下" |
| `【应用】` | 应用场景/实例 | "例如"、"应用"、"实际中"、"案例" |
| `【方法】` | 解题方法/算法 | "方法"、"步骤"、"算法"、"流程" |
| `【图表】` | 图示/表格 | 含图表、示意图、流程图 |

## 三、知识点关联映射

### 前置知识标注
- 识别知识点之间的依赖关系
- 标注：`前置：{依赖的知识点}`

### 跨章节关联
- 同一概念在不同章节出现时，标注：`关联：第X章 {概念}`
- 递进关系标注：`进阶：从{基础概念}到{高级概念}`

## 四、特殊内容处理

### 例题处理
- 例题单独归类为 `【应用】`
- 提取例题考查的知识点
- 标注解题方法标签

### 习题/思考题处理
- 不纳入知识清单主体
- 可作为考点提示的参考来源

### 图表处理
- 图表内容用文字描述其核心信息
- 标注：`（见原图/原表）`
