# Distillation Summary

CET Skill is based on structural analysis of 2015–2025 CET-4 and CET-6 materials. The analysis focuses on metadata, topic trends, task mechanics, difficulty patterns, scoring dimensions, and answer-feedback workflows.

## Key Patterns

- CET-4: shorter texts, clearer logic, more familiar topics, higher-frequency vocabulary.
- CET-6: denser information, more abstract topics, stronger inference requirements, more nuanced distractors.
- Writing: high-frequency topic directions, delayed model essay unless requested.
- Translation: Chinese-to-English transfer, standard and higher-scoring versions after user attempt.
- Reading: banked cloze, paragraph matching, and careful reading with complete CET-style item structure.
- Listening: scene-based scripts, delayed answer reveal, cue and distractor diagnosis.
