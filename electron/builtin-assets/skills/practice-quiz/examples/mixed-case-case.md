# 混合需求出题案例

## 用户输入

> 给我出一套线性代数矩阵部分的练习，要有选择填空和计算

## 处理过程

### 1. 解析需求
- 科目：线性代数
- 章节：矩阵
- 题型：混合（选择 + 填空 + 计算）
- 数量：未指定，按默认分配
- 难度：未指定，默认中等

### 2. 题型分配
按默认比例：
- 选择题：8 道
- 填空题：5 道
- 计算题：3 道

### 3. 知识点覆盖
矩阵章节核心知识点分配：
- 选择题覆盖：矩阵运算规则、逆矩阵条件、秩的概念等
- 填空题覆盖：行列式计算、矩阵性质等
- 计算题覆盖：矩阵求逆、矩阵秩的计算、线性方程组求解

### 4. 生成输出
先输出全部题目（选择 → 填空 → 计算），再统一输出答案解析。
