# 难度校准标准

## 三级难度定义

### 🟢 基础（Easy）
**目标**：检验基本概念和公式记忆

| 维度 | 标准 |
|------|------|
| 知识层级 | 识记、理解（布鲁姆分类法前两层） |
| 题目特征 | 直接套用公式、复述定义、简单识别 |
| 解题步骤 | 1-2 步 |
| 预期正确率 | 80%以上学生能做对 |

**典型题型**：
- "下列关于 xxx 的说法，正确的是？"
- "xxx 的定义是____。"
- "判断：xxx 一定成立。"

### 🟡 中等（Medium）
**目标**：检验理解深度和基本应用能力

| 维度 | 标准 |
|------|------|
| 知识层级 | 应用、分析（布鲁姆分类法中间两层） |
| 题目特征 | 需要理解原理后应用、涉及变形或组合 |
| 解题步骤 | 2-4 步 |
| 预期正确率 | 50%-80% 学生能做对 |

**典型题型**：
- "已知 xxx，求 xxx。"
- "比较 xxx 和 xxx 的异同。"
- "解释为什么 xxx 的情况下要用 xxx 方法。"

### 🔴 拔高（Hard）
**目标**：检验综合应用和分析能力

| 维度 | 标准 |
|------|------|
| 知识层级 | 评价、创造（布鲁姆分类法最高两层） |
| 题目特征 | 综合多个知识点、需要推导证明、开放性分析 |
| 解题步骤 | 4 步以上 |
| 预期正确率 | 50%以下学生能做对 |

**典型题型**：
- "证明：xxx。"
- "综合运用 xxx 和 xxx，分析以下问题。"
- "设计一个方案解决 xxx 问题。"

## 难度出题比例建议

| 场景 | 🟢 基础 | 🟡 中等 | 🔴 拔高 |
|------|---------|---------|---------|
| 日常练习 | 50% | 40% | 10% |
| 章节测验 | 40% | 40% | 20% |
| 期中/期末模拟 | 30% | 50% | 20% |
| 竞赛/考研 | 10% | 40% | 50% |

## 用户指定难度时的处理

- 用户说"简单/基础" → 全部 🟢
- 用户说"中等/正常" → 🟢30% + 🟡50% + 🔴20%
- 用户说"难/拔高" → 🟡40% + 🔴60%
- 用户说"模拟考试" → 按"期中/期末模拟"比例
