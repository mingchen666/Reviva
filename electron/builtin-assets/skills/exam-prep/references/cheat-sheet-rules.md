# 速记页生成规则

## 一、信息密度原则

### 压缩率目标
- 原始教材：100%
- 知识大纲：30-40%
- 速记页：5-10%

### 压缩方法
1. **去除非核心内容**：删除例子、背景介绍、历史沿革
2. **合并相似概念**：将同类知识点归入一行
3. **使用缩写和符号**：→ ∴ ∵ ≈ ≠ ≥ ≤ 等
4. **关键词替代句子**：用词组替代完整句子
5. **表格替代段落**：用表格结构化信息

## 二、速记页结构

### 必含部分
1. **核心概念速查表**（占 40%）
2. **公式/定理速查**（占 30%）
3. **易混对比**（占 15%）
4. **答题关键句**（占 15%）

### 可选部分（根据科目添加）
- 时间线/流程图（文科）
- 代码模板（计算机）
- 实验步骤（理工科）

## 三、排版要求

### 一页纸约束
- 控制在 A4 纸一页以内（约 80 行）
- 使用 4-5 列表格节省空间
- 关键词加粗，易错点标红（用 ⚠️ 标记）

### 可扫描性
- 每个知识点独立成行
- 不需要连续阅读，可以跳跃浏览
- 重要程度用符号标记：⭐⭐⭐

## 四、信息来源优先级

1. 老师划的重点（最高优先级）
2. 错题本中的高频知识点
3. 知识大纲中的 ⭐⭐⭐ 级别
4. 教材中的核心定义和公式
