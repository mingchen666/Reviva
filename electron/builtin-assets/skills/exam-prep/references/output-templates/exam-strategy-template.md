## 一、各题型答题模板

### 选择题
**解题步骤：**
1. 读题，圈出关键词
2. 排除明显错误选项
3. 在剩余选项中选择最佳答案
4. 不确定时用排除法

**时间控制：** 每题 1-2 分钟，共 {X} 分钟

### 填空题
**解题步骤：**
1. 读题，明确要求填写什么
2. 回忆相关公式/概念
3. 代入计算或直接写出答案
4. 检查答案的合理性

**时间控制：** 每题 1-2 分钟，共 {X} 分钟

### 简答题
**答题框架：**
1. 先写定义/概念（1-2 句）
2. 再写要点（分条列出）
3. 最后举例或补充说明

**得分技巧：**
- 分条作答，每条一个要点
- 关键词要突出
- 写不全也要写会的部分

**时间控制：** 每题 5-8 分钟，共 {X} 分钟

### 计算题
**解题步骤：**
1. 写出已知条件
2. 列出所用公式
3. 代入计算（过程要详细）
4. 写出结论（带单位）

**得分技巧：**
- 即使不会做，也要写公式（有步骤分）
- 计算过程要清晰，不要跳步
- 最后检查单位和符号

**时间控制：** 每题 10-15 分钟，共 {X} 分钟

## 二、考场策略

### 时间分配
| 题型 | 建议用时 | 实际用时 |
|------|----------|----------|
| 选择题 | {X} 分钟 | |
| 填空题 | {X} 分钟 | |
| 判断题 | {X} 分钟 | |
| 简答题 | {X} 分钟 | |
| 计算题 | {X} 分钟 | |
| 检查 | 10 分钟 | |

### 答题顺序
1. 先做会做的，跳过不会的
2. 选择题和填空题先做完（拿基础分）
3. 再做简答题和计算题
4. 最后回来做跳过的题
5. 最后 10 分钟检查

### 应急策略
- **选择题卡住**：排除法，至少排除 1-2 个选项
- **简答题想不起来**：写关键词，能拿几分是几分
- **计算题卡住**：先写公式，再尝试推导
- **时间不够**：优先做分值高的题

## 三、心态调整

- 考前 5 分钟：深呼吸 3 次，告诉自己"我已经准备好了"
- 遇到难题：先跳过，不要影响后面的发挥
- 遇到简单题：不要大意，仔细审题
- 最后检查：重点检查选择题和计算题的结果
