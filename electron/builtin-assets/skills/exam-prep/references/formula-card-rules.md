# 公式卡生成规则

## 一、公式选取标准

### 必选公式
- 教材中标注为"重要公式"的
- 课后习题反复使用的
- 老师明确要求记忆的

### 可选公式
- 推导过程中间结果（方便快速查阅）
- 特殊情况的简化公式

### 排除公式
- 仅用于推导，不直接使用的
- 已经内化为基本运算的（如加减乘除）

## 二、公式卡格式

### 每条公式包含
| 字段 | 说明 | 示例 |
|------|------|------|
| 公式 | 数学表达式 | `sin²x + cos²x = 1` |
| 名称 | 公式名称 | 勾股恒等式 |
| 适用条件 | 使用前提 | 对任意 x 成立 |
| 典型应用 | 常见使用场景 | 化简三角表达式 |
| 易错提醒 | 常见错误 | 别和 sin2x 混淆 |

### 排版规则
- 每章独立一个区块
- 公式用代码块格式（等宽字体，便于对齐）
- 适用条件用斜体或括号标注
- 易错提醒用 ⚠️ 标记

## 三、易错公式标注

### 常见易错类型
1. **记混型**：相似公式搞混（如 sin(A+B) vs sinA+sinB）
2. **条件型**：忽略适用条件（如洛必达法则的 0/0 型）
3. **符号型**：正负号记错
4. **系数型**：系数记错（如 1/2 vs 2）

### 标注格式
```
⚠️ 易错：{错误写法}
✅ 正确：{正确写法}
区别：{为什么容易错}
```
