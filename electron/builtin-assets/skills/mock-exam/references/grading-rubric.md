# 评分标准与扣分规则

## 一、各题型评分标准

### 选择题
- 正确：满分（2 分）
- 错误：0 分
- 多选/未选：0 分

### 填空题
- 完全正确：满分（2 分）
- 部分正确（多空题）：按空给分
- 答案等价但表述不同：给满分（如 0.5 = 1/2）

### 判断题
- 正确：满分（1 分）
- 错误：0 分

### 简答题
按要点给分：
| 情况 | 给分 |
|------|------|
| 所有要点齐全，表述准确 | 满分 |
| 核心要点齐全，表述不够精确 | 80% |
| 部分要点遗漏 | 按比例给分 |
| 要点全错但有相关表述 | 10-20%（辛苦分） |
| 空白或完全无关 | 0 分 |

### 计算题
按步骤给分：
| 步骤 | 分值占比 |
|------|----------|
| 已知条件整理 | 10% |
| 公式选用正确 | 20% |
| 代入计算过程 | 40% |
| 最终结果正确 | 20% |
| 单位/结论完整 | 10% |

扣分规则：
- 公式正确但代入错误：扣代入部分分
- 过程正确但结果错误：扣结果部分分（可能计算失误）
- 结果正确但无过程：最多给 50%（无法判断是否理解）
- 使用了超纲方法：扣方法部分分

### 综合题
- 分步骤评分，每步独立给分
- 如果后续步骤依赖前步结果，前步错则后续步骤的"代入"部分不重复扣分

## 二、特殊情况处理

### 半对处理
- 答案部分正确：给 50% 分值
- 多选题选对一个选错一个：0 分
- 填空题写了两个答案（一个对一个错）：0 分

### 涂改处理
- 以最终清晰的答案为准
- 如果无法判断哪个是最终答案：0 分

### 超出答题区域
- 正常批改，不因超出区域扣分

## 三、批改报告规范

### 错题标注格式
```
❌ 错误（0分）| 你的答案：{答案} | 正确答案：{答案}
```

### 半对标注格式
```
⚠️ 半对（{X}分/{满分}）| 答案不完整，缺少 {内容}
```

### 正确标注格式
```
✅ 正确（{满分}分）
```
