# 完整模拟考试流程案例

## 用户输入

> 给我出一套高数期末模拟卷

## 完整流程

### 阶段一：出卷

1. 解析需求：科目=高等数学，范围=全部，时长=120分钟
2. 参考 `paper-structure-rules.md` 设计试卷结构
3. 使用 `exam-paper-template.md` 生成试卷

输出试卷后提示：
```
模拟卷已生成，满分 100 分，考试时间 120 分钟。
请在规定时间内完成，然后将答案提交给我。
```

### 阶段二：等待作答

学生在 120 分钟后提交答案。

### 阶段三：批改

1. 逐题对照答案批改
2. 按 `grading-rubric.md` 评分
3. 使用 `grading-report-template.md` 输出批改报告

### 阶段四：分析

1. 使用 `score-analysis-template.md` 输出分数分析
2. 识别薄弱知识点
3. 给出改进建议
