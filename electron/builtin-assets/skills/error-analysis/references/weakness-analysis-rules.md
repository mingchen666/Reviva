# 薄弱点归类与分析规则

## 一、知识点映射

### 映射方法
1. 从错题中提取考查的知识点
2. 将知识点归入章节/知识模块
3. 统计每个知识点的错误次数

### 映射表格式
| 错题编号 | 考查知识点 | 所属章节 | 错因类型 |
|----------|-----------|----------|----------|
| #1 | {知识点} | {章节} | {错因} |
| #2 | {知识点} | {章节} | {错因} |

## 二、薄弱度计算

### 单个知识点薄弱度
```
薄弱度 = 错误次数 × 错因权重
```

错因权重：
| 错因 | 权重 | 理由 |
|------|------|------|
| 概念不清 | 3.0 | 根本性问题，影响后续学习 |
| 知识断层 | 3.0 | 需要补前置知识 |
| 公式误用 | 2.5 | 需要重新记忆 |
| 方法不当 | 2.0 | 需要练习方法选择 |
| 审题不细 | 1.5 | 需要训练审题习惯 |
| 计算失误 | 1.0 | 需要提高计算准确性 |
| 粗心大意 | 0.5 | 需要养成检查习惯 |

### 薄弱点排序
按薄弱度降序排列，优先级最高排最前。

## 三、薄弱点报告结构

### 报告内容
1. **错因统计**：各错因类型出现次数
2. **知识点排序**：薄弱知识点从高到低
3. **错误模式**：反复出现的同类错误
4. **改进建议**：每个薄弱点的针对性建议

### 改进建议生成规则

| 薄弱类型 | 建议 |
|----------|------|
| 概念不清 | 重读教材相关章节，制作概念对比卡 |
| 公式误用 | 整理公式卡，标注适用条件 |
| 计算失误 | 每天做 5 道计算题，养成验算习惯 |
| 审题不细 | 做题前圈出关键词，列出已知条件 |
| 方法不当 | 整理题型-方法对照表，做专项练习 |
| 知识断层 | 补学前置知识，再回到当前内容 |
| 粗心大意 | 做完检查，用不同方法验算 |
