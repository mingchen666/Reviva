# 错因分类与错误模式识别规则

## 一、错因分类体系

| 错因类型 | 标签 | 说明 | 识别特征 |
|----------|------|------|----------|
| 概念不清 | `🏷 概念` | 对定义、定理理解有误 | 答案与正确概念矛盾 |
| 公式误用 | `🏷 公式` | 记错公式或用错条件 | 公式形式正确但细节错误 |
| 计算失误 | `🏷 计算` | 运算过程出错 | 思路对，结果错 |
| 审题不细 | `🏷 审题` | 漏看条件、理解偏差 | 答非所问或遗漏条件 |
| 方法不当 | `🏷 方法` | 解题思路/方法选择错误 | 方法可行但非最优或根本行不通 |
| 知识断层 | `🏷 断层` | 前置知识缺失导致无法解题 | 卡在某一步，无法继续 |
| 粗心大意 | `🏷 粗心` | 非知识性失分 | 简单题做错、抄错数字等 |

## 二、错因诊断流程

### Step 1：还原解题思路
- 如果用户提供了作答过程，直接分析
- 如果只给了错误答案，尝试推断可能的错误思路

### Step 2：定位出错环节
- 是理解题意就错了？→ 审题
- 是公式/概念记错了？→ 概念/公式
- 是方法选错了？→ 方法
- 是中间计算错了？→ 计算
- 是完全不会做？→ 断层

### Step 3：确定根本原因
- 表面原因（如"算错了"）→ 深挖为什么算错（是公式记错还是运算失误）
- 关联原因（如"不会做"）→ 深挖缺了什么前置知识

## 三、错误模式识别

### 模式定义
同一错因类型在 3 道以上题目中出现，构成**错误模式**。

### 模式等级
| 等级 | 出现次数 | 提示 |
|------|----------|------|
| 初现 | 1-2 次 | 正常，个别失误 |
| 警告 | 3 次 | ⚠️ 需要关注 |
| 严重 | 5 次以上 | 🔴 必须专项突破 |

### 常见错误模式

**模式 A：概念混淆型**
- 特征：相似概念反复搞混
- 例：连续 vs 可导、充分 vs 必要
- 对策：制作对比表，强化区分

**模式 B：公式记忆型**
- 特征：公式记错或记不全
- 例：漏记适用条件、记混相似公式
- 对策：公式卡 + 适用条件标注

**模式 C：计算粗心型**
- 特征：思路对但结果错
- 例：符号错误、抄错数字
- 对策：计算后验算习惯

**模式 D：方法选择型**
- 特征：解题方法选错或不会选
- 例：该用洛必达却用等价无穷小
- 对策：题型-方法对照表
