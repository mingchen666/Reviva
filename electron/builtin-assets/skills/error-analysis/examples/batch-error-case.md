# 批量错题整理案例

## 用户输入

> 帮我整理一下这几道错题：
> 1. 求 lim(x→0) sin(3x)/x，我答 0，答案是 3
> 2. 求 lim(x→∞) (1+2/x)^x，我答 1，答案是 e²
> 3. 求导 y=x²sinx，我答 2xsinx，答案是 2xsinx+x²cosx
> 4. 求 lim(x→0) (1-cosx)/x²，我答 0，答案是 1/2

## 处理过程

### 1. 逐题分析

| # | 题目 | 错因 | 知识点 |
|---|------|------|--------|
| 1 | sin(3x)/x | 公式误用 | 重要极限 |
| 2 | (1+2/x)^x | 公式误用 | 重要极限 |
| 3 | x²sinx 求导 | 概念不清 | 乘法求导法则 |
| 4 | (1-cosx)/x² | 公式误用 | 等价无穷小 |

### 2. 识别模式
- 公式误用出现 3 次 → ⚠️ 警告级别
- 涉及重要极限/等价无穷小的知识点反复出错
- 第3题是独立的求导法则问题

### 3. 输出错题本 + 错误模式分析

```
## 错误模式分析

### 高频错因统计
| 错因类型 | 出现次数 | 涉及知识点 |
|----------|----------|------------|
| 🏷 公式 | 3次 | 重要极限、等价无穷小 |
| 🏷 概念 | 1次 | 乘法求导法则 |

### 错误模式提醒
⚠️ **重要极限公式误用**：连续 3 道极限题都用错公式（出现 3 次）
→ 建议：重新整理两个重要极限的公式卡，标注适用条件
```
