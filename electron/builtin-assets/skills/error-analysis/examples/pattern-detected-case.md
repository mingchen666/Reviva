# 发现错误模式案例

## 背景

用户已经提交过 8 道错题，本次又提交 2 道新的。

## 用户输入

> 再加两道：
> 9. 求导 y=ln(sinx)，我答 cosx，答案是 cosx/sinx=cotx
> 10. 求导 y=ln(x²+1)，我答 1/(x²+1)，答案是 2x/(x²+1)

## 处理过程

### 1. 分析新错题
- #9：复合函数求导，忘了链式法则 → 🏷 方法
- #10：复合函数求导，忘了链式法则 → 🏷 方法

### 2. 更新统计
回顾之前的 8 道错题，发现：
- #3（乘法求导法则）→ 🏷 概念
- #9（链式法则）→ 🏷 方法
- #10（链式法则）→ 🏷 方法

**求导相关错题已有 3 道**，触发错误模式检测。

### 3. 输出更新

```
## 新增错题

### 错题 #9
...

### 错题 #10
...

---

## 错误模式更新

### 新检测到的模式
🔴 **求导法则不熟练**：求导相关错题已达 3 道
- #3：乘法求导法则遗漏
- #9：链式法则遗漏
- #10：链式法则遗漏

→ 建议：专项复习求导法则，重点练习复合函数求导
→ 已生成 3 道针对性巩固题（见下方）

---
```
