# 单道错题分析案例

## 用户输入

> 这道题我做错了：求 lim(x→0) sin(3x)/x，我写的答案是 0，正确答案是 3

## 处理过程

### 1. 读取与识别
- 题目：求极限 lim(x→0) sin(3x)/x
- 用户答案：0
- 正确答案：3
- 未提供作答过程，尝试推断错误思路

### 2. 错因推断
用户答案为 0，可能的错误思路：
- 直接代入 x=0，得到 sin(0)/0 = 0/0，不知道怎么处理
- 或者错误地认为 sin(3x) → 0，所以结果是 0

这属于**公式误用**或**知识断层**：没有掌握重要极限 lim(x→0) sin(x)/x = 1。

### 3. 生成输出

```
## 错题 #1

- **题目**：求 lim(x→0) sin(3x)/x
- **我的答案**：0
- **正确答案**：3
- **错因分析**：未正确应用重要极限公式。sin(3x) 在 x→0 时与 3x 等价，
  因此 sin(3x)/x = 3·sin(3x)/(3x) → 3·1 = 3。
  直接代入得到 0/0 是不定式，不能直接得出 0。
- **考查知识点**：两个重要极限、等价无穷小
- **错因标签**：🏷 公式
- **纠正要点**：看到 sin(□x)/x 型极限，化为 □·sin(□x)/(□x) → □
```
