Mermaid 语法帮助

1. 思维导图 (mindmap)

用于层级概念展示，从中心向外扩展：

mindmap
  root((中心主题))
    分支一
      子节点1
      子节点2
    分支二
      子节点3

节点形状：((圆)) 圆形根节点，(圆角) 圆角矩形，[方括号] 默认形状

2. 流程图 (flowchart)

用于流程和算法展示：

flowchart TD
  A[开始] --> B{判断}
  B -->|是| C[执行A]
  B -->|否| D[执行B]
  C --> E[结束]
  D --> E

方向：TD 上到下，LR 左到右，RL 右到左

节点：[矩形]，{菱形}，((圆形))，[/平行四边形/]

3. 类图 (classDiagram)

用于面向对象和CS概念：

classDiagram
  Animal <|-- Duck
  Animal : +String name
  Animal : +makeSound()
  class Duck {
    +swim()
  }

关系：<|-- 继承，*-- 组合，o-- 聚合，..> 依赖

4. 时序图 (sequenceDiagram)

用于交互流程展示：

sequenceDiagram
  participant A as 客户端
  participant B as 服务器
  A->>B: 发送请求
  B-->>A: 返回响应

箭头：->> 实线，-->> 虚线，-x 异步

5. 样式自定义

使用 classDef 定义样式类：

classDef highlight fill:#ff6b6b,stroke:#333,stroke-width:2px
classDef info fill:#4ecdc4,stroke:#333,stroke-width:2px
A:::highlight
B:::info

6. 实用技巧

  • 使用 &nbsp; 或引号包裹含特殊字符的文本
  • 用 subgraph 对节点分组
  • 用 click 添加链接交互
  • 长文本用 <br/> 换行
  • 中文文本建议用引号包裹避免解析问题
