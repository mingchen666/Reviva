## 考试特化出题规则

### 中考文科

- **难度控制**：以1-2级为主，少量3级
- **记忆要求**：重大事件的时间、人物、结果必须准确记忆
- **情境设置**：以课本知识点为主，不引入课外材料
- **选项设计**：4个选项差异明显，干扰项为常见知识混淆
- **时间线考查**：重大事件排序、朝代更替顺序
- **禁止出现**：学术争议性话题、复杂史料批判、多层次论证
- **简答题重点**：原因概括（1-2点）、特点归纳、简单评价

### 高考文科

- **难度分布**：1级10% + 2级40% + 3级35% + 4级15%
- **材料题特征**：
  - 材料来源必须真实可考（标注出处）
  - 材料长度100-200字，考研200-400字
  - 每则材料聚焦一个观点/事实
  - 问题必须"根据材料"作答，强调论从史出
- **选择题特征**：
  - 选项间逻辑关联，非随机排列
  - 至少1个选项考查概念理解而非知识记忆
  - 干扰项"看似合理但材料不支持"
- **论述题特征**：
  - 设问由浅入深：(1)概括→(2)分析→(3)评价
  - 要求有论点、论据、论证的完整结构
  - 鼓励多角度分析，但需有主次
- **价值导向**：坚持正确价值观，允许合理多元解读

### 考研文科

- **学术深度**：涉及学术观点争鸣，要求辨析不同学派立场
- **史料批判**：考查史料来源、立场、可靠性的批判能力
- **理论素养**：需运用理论框架分析问题（如现代化理论、世界体系理论等）
- **难度分布**：1级5% + 2级20% + 3级40% + 4级35%
- **论述要求**：
  - 需引证学术成果（标注学者/著作）
  - 区分一手史料和二手研究
  - 论证要有理论支撑，不能仅靠史实罗列
- **方法论考查**：史学方法、比较方法、计量方法等
- **开放度**：允许不同学术观点的合理表达，但论证必须严密

## 干扰项设计指南

### 文科学科干扰项来源

#### 1. 时空错位型

将事件/制度/政策放置到错误的时代。

| 正确配对 | 干扰配对 | 错误类型 |
|---------|---------|---------|
| 科举制始于隋唐 | 科举制始于西汉 | 时代前移近800年 |
| 行省制度创立于元朝 | 行省制度创立于宋朝 | 朝代错位 |
| 《权利法案》颁布于1689年 | 《权利法案》颁布于1789年 | 与法国大革命年代混淆 |
| 罗斯福新政实施于1930年代 | 罗斯福新政实施于一战后 | 时代背景错误 |
| 改革开放始于1978年 | 改革开放始于1966年 | 与文革混淆 |

#### 2. 因果倒置型

混淆原因与结果、前提与产物。

| 正确因果 | 干扰因果 | 说明 |
|---------|---------|------|
| 工业革命→城市化 | 城市化→工业革命 | 颠倒因果，城市化是工业革命的结果 |
| 甲午战败→维新运动兴起 | 维新运动→甲午战败 | 颠倒因果，战败是变法的原因 |
| 经济危机→罗斯福新政 | 罗斯福新政→经济危机 | 新政是应对危机的结果 |
| 供需失衡→价格变动 | 价格变动→供需失衡 | 颠倒基本经济逻辑 |

#### 3. 以偏概全型

从个别案例过度推广为普遍结论。

| 合理表述 | 过度推广 | 说明 |
|---------|---------|------|
| 科举制扩大了统治基础 | 科举制实现了社会公平 | 扩大≠实现，科举仍有阶层局限 |
| 工业革命推动了社会进步 | 工业革命使所有人生活改善 | 忽视工人阶级的苦难 |
| 英国1832年改革扩大了选举权 | 1832年改革实现了民主 | 65%男性+全部妇女仍无选举权 |
| 市场经济促进了效率 | 市场经济解决了所有经济问题 | 忽视市场失灵 |

#### 4. 现代投射型

将现代价值观/标准投射到历史分析中。

| 历史事实 | 现代投射 | 说明 |
|---------|---------|------|
| 古代科举制的主要功能是加强中央集权 | 古代科举制的主要功能是促进教育公平 | 用现代教育观评价古代选官制度 |
| 洋务运动在维护体制的前提下学习西方技术 | 洋务运动应该彻底学习西方制度 | 用后见之明要求历史人物超前行动 |
| 宗法制在古代起维护社会秩序的作用 | 宗法制是纯粹的压迫制度 | 忽视制度的历史合理性和功能 |
| 重农抑商政策在农业社会有其合理性 | 重农抑商是完全错误的政策 | 脱离时代背景做价值判断 |

#### 5. 概念混淆型

相似但不同的政治/经济概念交叉混淆。

| 概念A | 概念B | 混淆点 |
|-------|-------|-------|
| 国体 | 政体 | 内容与形式的关系 |
| 财政政策 | 货币政策 | 工具和制定主体不同 |
| 绝对优势 | 比较优势 | 斯密vs李嘉图，条件强弱不同 |
| 名义GDP | 实际GDP | 是否剔除通胀因素 |
| 需求量变动 | 需求变动 | 沿曲线移动vs曲线整体移动 |
| 经济增长 | 经济发展 | 量变vs质变 |
| 专制 | 专政 | 剥削阶级独裁vs人民民主 |

### 干扰项设计流程

1. **确定正确答案**：先明确正确观点及其依据
2. **列举典型误解**：回忆/查找学生常见误解和学术争议
3. **生成干扰项**：按上述5种类型生成候选干扰项
4. **学术合理性检查**：每个干扰项必须有"学术合理性"——不能是明显荒谬的表述
5. **材料一致性检查**：材料分析题的干扰项必须"看似有材料依据但经不起推敲"
6. **区分度优化**：高考干扰项区分度较大，考研干扰项区分度较小
