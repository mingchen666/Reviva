# 数学出题指南

> 本文档为 AI 出题系统提供数学学科的出题规范，覆盖中考、高考、考研三大考试，涵盖知识点映射、题型模板、干扰项设计及完整 JSON 示例。

---

## 1. 学科-考试矩阵

| 考试层级 | 核心知识域 | 题型分布 | 难度范围 | 总分/时长 |
|---------|-----------|---------|---------|----------|
| **中考** | 代数(方程/不等式/函数)、几何(三角形/圆/四边形)、统计与概率 | 选择题 30% + 填空题 20% + 解答题 50% | ⭐ ~ ⭐⭐⭐（基础为主，压轴轴⭐⭐⭐） | 120分 / 120min |
| **高考** | 函数与导数、解析几何、概率统计、数列、三角函数、立体几何 | 选择题 40% + 填空题 15% + 解答题 45% | ⭐ ~ ⭐⭐⭐⭐⭐（区分度集中在⭐⭐⭐⭐） | 150分 / 120min |
| **考研** | 高等数学(微积分)、线性代数、概率论与数理统计 | 选择题 32% + 填空题 16% + 解答题 52% | ⭐⭐ ~ ⭐⭐⭐⭐⭐（计算与证明并重） | 150分 / 180min |

**难度标定说明：**
- ⭐：直接套用公式/定义，一步可得
- ⭐⭐：需1-2步推理，涉及单一知识点
- ⭐⭐⭐：需多步推理，可能涉及2个知识点交叉
- ⭐⭐⭐⭐：需构造性思维或复杂计算，多知识点综合
- ⭐⭐⭐⭐⭐：需创新性思路或深度证明，压轴级别

## 3. 出题模板

### 模板A：概念辨析选择题

**适用知识点：** 所有知识域中的概念性内容（定义、性质、判定条件）

**难度范围：** ⭐ ~ ⭐⭐⭐

**题目结构：**
```
题干：给出一个数学命题/关系/条件，要求判断正确性或选择等价表述
选项：4个选项，其中1个正确，3个干扰
```

**干扰项设计原则：**
1. **条件缺失型**：正确命题去掉某个前提条件（如"连续"→"可导"去掉连续前提）
2. **方向反转型**：充分条件与必要条件互换
3. **概念混淆型**：相似但不同的概念替换（如"收敛"→"绝对收敛"）
4. **特殊代替一般型**：用特殊情形的结论代替一般结论

**hint层级设计：**
- hint1：指出需要回顾的核心概念/定义
- hint2：给出关键条件的提示
- hint3：指出干扰项的具体错误所在

**commonErrors设计：**
- 记录选择各干扰项的典型原因
- 标注对应的认知误区

### 模板B：计算填空题

**适用知识点：** 所有知识域中的计算型内容（求值、化简、求解）

**难度范围：** ⭐⭐ ~ ⭐⭐⭐⭐

**题目结构：**
```
题干：给出具体数学对象（函数、矩阵、方程等），要求计算特定结果
答案：数值或简洁数学表达式
```

**干扰项设计原则（填空题转化为选择题时）：**
1. **计算步骤错误型**：在某一步计算出错得到的错误结果
2. **符号错误型**：正负号搞反
3. **公式记错型**：用错公式但计算过程合理
4. **遗漏步骤型**：忘记最后一步（如换元后忘记回代）

**hint层级设计：**
- hint1：提示应使用的公式/方法
- hint2：给出关键中间步骤
- hint3：展示完整解题路径的关键节点

**commonErrors设计：**
- 记录典型错误答案及其产生原因
- 标注出错的计算步骤

**JSON示例（⭐⭐⭐级）：**

### 模板C：综合解答题

**适用知识点：** 多知识点交叉、需要多步推理的综合性内容

**难度范围：** ⭐⭐⭐ ~ ⭐⭐⭐⭐⭐

**题目结构：**
```
题干：给出一个较复杂的数学情境，包含2-3个小问
小问设计：递进式（第1问铺垫，第2问深入，第3问综合/证明）
```

**干扰项设计原则（解答题不设选项，但需设计评分节点）：**
1. **关键步骤评分点**：每个重要推理/计算步骤设为评分节点
2. **常见错误路径**：预判学生可能走错的路径及对应扣分
3. **部分得分设计**：即使最终答案错误，中间正确步骤也应得分

**hint层级设计：**
- hint1：提示第1问的入手方向
- hint2：提示第1问的关键步骤 + 第2问的入手方向
- hint3：给出各问的核心解题思路

**commonErrors设计：**
- 按小问分别记录常见错误
- 标注错误产生的认知根源

**JSON示例（⭐⭐⭐⭐⭐级）：**

## 4. 考试特化出题规则

### 4.1 中考出题规则

| 规则项 | 具体要求 |
|-------|---------|
| **难度上限** | 最高 ⭐⭐⭐⭐（仅压轴题最后一问），主体 ⭐⭐~⭐⭐⭐ |
| **计算量限制** | 不超过3步运算；数字尽量取整数或简单分数；避免复杂根式运算 |
| **知识范围** | 严格限制在课标范围内，不超纲；不涉及高等数学概念 |
| **特殊题型** | 规律探究题、操作探究题、阅读理解题（新定义题）、动态几何题 |
| **几何要求** | 证明题步骤不超过5步；辅助线不超过2条；不要求复杂构造 |
| **函数要求** | 仅限一次函数、二次函数、反比例函数；不涉及分段函数（除新定义题） |
| **应用题** | 情境贴近生活；数据合理；不出现负数解的物理量 |
| **压轴题结构** | 通常3问：第1问求解析式（⭐⭐），第2问求坐标/面积（⭐⭐⭐），第3问存在性问题（⭐⭐⭐⭐） |

**中考特殊题型模板——新定义题：**
```
题干结构：
1. 给出一个新运算/新概念的定义
2. 第1问：用新定义计算/判断简单情形（⭐⭐）
3. 第2问：用新定义解决稍复杂问题（⭐⭐⭐）
4. 第3问：结合新定义与已有知识综合探究（⭐⭐⭐⭐）
```

### 4.2 高考出题规则

| 规则项 | 具体要求 |
|-------|---------|
| **难度上限** | 最高 ⭐⭐⭐⭐⭐（压轴题最后一问），主体 ⭐⭐⭐~⭐⭐⭐⭐ |
| **计算量限制** | 允许较复杂计算，但应避免纯机械运算；关键步骤应有思维含量 |
| **知识范围** | 课标+合理延伸；导数应用可涉及不等式证明；解析几何可涉及韦达定理 |
| **特殊题型** | 多选题（新高考）、开放性填空题、结构不良题（条件可选取） |
| **导数大题** | 标准结构：讨论单调性→求极值/最值→证明不等式；含参讨论为必考 |
| **解析几何大题** | 标准结构：求方程→弦长/面积→定点/定值/存在性；计算量需控制 |
| **概率统计大题** | 情境应用为主；可能涉及决策问题；数据量适中 |
| **多选题规则** | 4个选项，正确选项2-3个；部分选对得2分，全对得5分，选错0分 |

**高考多选题干扰项设计：**
- 每个选项独立判断，不互斥
- 至少1个选项为"看似正确但条件不足"型
- 至少1个选项为"特殊情形成立但一般不成立"型

### 4.3 考研出题规则

| 规则项 | 具体要求 |
|-------|---------|
| **难度上限** | 最高 ⭐⭐⭐⭐⭐，主体 ⭐⭐⭐~⭐⭐⭐⭐⭐ |
| **计算量限制** | 允许较大计算量；证明题要求严格逻辑；计算题要求准确结果 |
| **知识范围** | 考研大纲为界；数学一最广（含曲线积分、曲面积分），数学三次之，数学二最窄 |
| **特殊题型** | 证明题（占比高）；综合应用题；反例构造题 |
| **高数大题** | 可能涉及：中值定理证明、级数审敛与求和、多元函数极值、二重积分计算 |
| **线代大题** | 可能涉及：含参方程组讨论、特征值综合应用、二次型标准化与正定性 |
| **概率大题** | 可能涉及：随机变量函数的分布、参数估计、假设检验 |
| **数学一二三差异** | 数学一：曲线/曲面积分、假设检验；数学二：无级数、无概率统计、无空间解析几何；数学三：经济应用、无曲面积分 |

**考研证明题出题要点：**
- 题目需有明确的证明目标
- 提供合理的证明路径（至少2条）
- 评分按逻辑步骤给分，不因方法不同而扣分
- 常见证明工具：中值定理、单调性分析、构造辅助函数、数学归纳法

## 5. 干扰项设计指南

### 5.1 数学学科通用干扰项类型

| 干扰项类型 | 描述 | 适用场景 | 示例 |
|-----------|------|---------|------|
| **条件缺失** | 正确结论去掉某个前提条件 | 概念辨析题 | "可导函数必连续" → 干扰项"函数必连续" |
| **方向反转** | 充分条件与必要条件互换 | 逻辑判断题 | "A是B的充分条件" → 干扰项"A是B的必要条件" |
| **概念替换** | 用相似但不同的概念替换 | 概念辨析题 | "收敛" → "绝对收敛"；"连续" → "可导" |
| **特殊代一般** | 特殊情形的结论推广到一般 | 性质判断题 | "正交矩阵的行列式为1" → 干扰项"行列式为1的矩阵正交" |
| **计算步骤错** | 在某一步出错但后续计算合理 | 计算题 | 求导时链式法则漏内层，后续计算正确 |
| **符号错误** | 正负号搞反 | 计算题 | 积分结果差一个负号 |
| **公式误用** | 用了形似但不同的公式 | 计算题 | 用错求导公式或积分公式 |
| **遗漏步骤** | 忘记最后一步或中间关键步骤 | 计算题 | 换元后忘记回代；忘记加常数C |
| **范围扩大/缩小** | 定义域/值域/取值范围出错 | 求解题 | 解集多包含或少包含边界值 |
| **方法错配** | 用了不适用当前情境的方法 | 综合题 | 交错级数用比值审敛法而非莱布尼茨判别法 |

### 5.2 各知识域特有干扰项

**微积分特有干扰项：**

| 干扰项 | 产生原因 | 典型表现 |
|-------|---------|---------|
| 极限不存在误判为极限为∞ | 混淆"极限不存在"与"趋于无穷" | lim sin(1/x) 误答为 ∞ |
| 洛必达法则滥用 | 不验证0/0或∞/∞型就使用 | 非未定式直接洛必达 |
| 积分上下限不变 | 定积分换元后忘记变换上下限 | ∫₀^{π/2} sin²xcosx dx 换元后仍用0和π/2 |
| 级数收敛半径端点遗漏 | 只求收敛半径不讨论端点 | R=1时未讨论x=±1 |
| 变限积分求导遗漏 | 忘记对上限函数求导 | d/dx ∫₀^{x²} f(t)dt = f(x²)（漏了2x） |

**线性代数特有干扰项：**

| 干扰项 | 产生原因 | 典型表现 |
|-------|---------|---------|
| |A+B| = |A|+|B| | 行列式性质与实数加法混淆 | 误以为行列式可分配 |
| (AB)⁻¹ = A⁻¹B⁻¹ | 逆矩阵顺序搞反 | 正确应为 B⁻¹A⁻¹ |
| r(AB) = r(A)·r(B) | 秩的性质与乘法混淆 | 正确应为 r(AB) ≤ min(r(A),r(B)) |
| 特征向量可以为零向量 | 忽略定义中的非零条件 | 零向量不是特征向量 |
| 实对称矩阵特征值必为正 | 混淆正定矩阵与实对称矩阵 | 实对称矩阵特征值为实数，不一定为正 |

**概率统计特有干扰项：**

| 干扰项 | 产生原因 | 典型表现 |
|-------|---------|---------|
| P(AB) = P(A)·P(B) | 不验证独立性就使用乘法公式 | 非独立事件误用乘法 |
| P(A|B) = P(B|A) | 条件概率方向搞反 | 混淆"已知B求A"与"已知A求B" |
| 置信水平=概率 | 误解置信区间的频率解释 | "参数有95%的概率落在置信区间内" |
| 小概率事件=不可能事件 | 混淆"几乎不可能"与"不可能" | 假设检验中拒绝H₀不等于H₀为假 |
| 正态分布直接查表 | 忘记标准化步骤 | 非标准正态直接用标准正态表 |

**中考/高中数学特有干扰项：**

| 干扰项 | 产生原因 | 典型表现 |
|-------|---------|---------|
| 二次函数顶点公式记错 | 符号或分母错误 | 顶点横坐标写成 b/2a 或 -b/a |
| 韦达定理符号错 | 两根之和/积的符号 | x₁+x₂ = -b/a 写成 b/a |
| 不等式乘负数不变号 | 运算规则遗忘 | 两边乘-1后不等号方向不变 |
| 解三角形多解遗漏 | 忽略钝角/锐角两种可能 | 已知两边及一对角时遗漏解 |
| 圆的切线判定与性质混淆 | 两个方向搞反 | "过切点的半径垂直于切线"与"过半径外端的垂线是切线"混淆 |

### 5.3 干扰项质量评估标准

| 评估维度 | 优秀标准 | 合格标准 | 不合格标准 |
|---------|---------|---------|-----------|
| **迷惑性** | 错选率 > 20% | 错选率 10%-20% | 错选率 < 10% |
| **诊断性** | 能精确定位认知误区 | 能定位到知识模块 | 无法定位问题 |
| **合理性** | 基于真实认知过程产生 | 有一定合理性 | 明显不合理/荒谬 |
| **独立性** | 各干扰项对应不同误区 | 部分重叠 | 高度重叠 |
| **均匀性** | 各干扰项选择率相近 | 有主有次但都>5% | 某项几乎无人选择 |
