# 题库出题指南 — 总则

本目录包含各学科的出题指南，指导AI生成高质量、结构化的练习题和测验题。

## 题目元数据规范

生成题目时，应在现有quiz-template的5种题型基础上添加以下元数据字段（全部可选，向后兼容）：

### 基础字段（quiz-template已有）

| 字段 | 必填 | 说明 |
|------|------|------|
| type | 是 | choice/fill/match/tf/order |
| difficulty | 是 | 1-4（⭐基础/⭐⭐应用/⭐⭐⭐综合/⭐⭐⭐⭐挑战） |
| question | 是 | 题目内容（支持\n换行） |
| answer | 是 | 答案（格式因题型而异） |
| explanation | 是 | 解析 |
| pitfall | 否 | 易错提醒 |

### 增强字段（practice-mode使用）

| 字段 | 必填 | 说明 |
|------|------|------|
| subject | 推荐 | 学科：数学/物理/化学/生物/英语/CS/文科 |
| topic | 推荐 | 知识点/章节，如"导数"、"力学-牛顿定律" |
| tags | 推荐 | 标签数组，如["导数","极值","高考"] |
| exam | 按需 | 适配考试：中考/高考/四级/六级/考研 |
| cognitiveLevel | 推荐 | 认知层级：1记忆/2理解/3应用/4分析/5综合 |
| hints | 推荐 | 3层渐进提示：[方向, 方法, 步骤] |
| commonErrors | 推荐 | 常见错误及原因：[{answer, reason}] |
| id | 间隔重复时必填 | 唯一ID，格式：{学科}-{主题}-{序号} |

### 增强字段示例

{

  "subject": "数学",

## AI出题总则

### 1. 质量标准

- **准确性**：题目、答案、解析必须正确无误
- **针对性**：每道题明确考查1-2个知识点
- **区分度**：干扰项要有区分力，不能太明显也不能太模糊
- **教育性**：解析要讲清"为什么"，而不只是给答案

### 2. 难度分级

| 级别 | 标识 | 认知要求 | 典型特征 |
|------|------|---------|---------|
| 1 | ⭐ | 记忆/理解 | 直接应用公式，1步推理 |
| 2 | ⭐⭐ | 理解/应用 | 需识别方法，2-3步推理 |
| 3 | ⭐⭐⭐ | 应用/分析 | 组合知识点，多步推理 |
| 4 | ⭐⭐⭐⭐ | 分析/综合 | 非标准情境，需要建模 |

### 3. 渐进提示设计

3层提示遵循"不替代思考"原则：

- **Level 1 方向提示**：指明思考方向，不给具体内容。如："想想导数的几何意义"
- **Level 2 方法提示**：点明应使用的方法。如："先求f'(x)找驻点，再用二阶导数判断"
- **Level 3 步骤提示**：给出接近完整的解题路径。如："f'(x)=3x²-3=0 → x=±1，f''(1)=6>0为极小"

### 4. 干扰项设计

好的干扰项来自真实的学生错误：

- **概念混淆**：把相关但不同的概念混在一起（如极大vs极小）
- **计算错误**：常见计算步骤出错的结果（如符号错误）
- **部分理解**：理解了一半的答案（如知道求导但忘了判断极值类型）
- **过度推广**：把特殊规律错误推广到一般情况
