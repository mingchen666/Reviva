# 知识图谱复习详细指南

## 核心理念

知识不是孤立的点，而是相互关联的网络。知识图谱复习的核心是**从全局到局部，从联系到细节**，帮助学生建立系统化的知识结构。

期末复习最忌"见树不见林"——只盯着零散的知识点，看不到它们之间的联系。知识图谱复习让学生先看到整片森林，再深入每棵树。

## 三阶段复习法

### 阶段1：构建全景图（全局视角）

1. **列出核心主题**：从课程大纲或教材目录出发
2. **画出知识地图**：展示主题之间的逻辑关系
3. **标记掌握程度**：用颜色/符号标注已掌握和未掌握

```
高数知识图谱

[极限] ──→ [连续] ──→ [可导] ──→ [可积]
  🟢        🟢        🟡        🔴

  [极限] ──→ [级数]
              🔴

图例：🟢已掌握  🟡部分掌握  🔴未掌握
```

4. **确定优先级**：先复习影响面大且未掌握的核心节点

### 阶段2：逐节点深入（局部深化）

对每个需要复习的节点：

1. **核心概念**：这个知识点到底是什么
2. **关键公式/定理**：需要记住什么
3. **与其他节点的联系**：为什么它在这个位置
4. **典型题型**：考试怎么考
5. **常见错误**：容易在哪里出错

### 阶段3：串联检验（综合测试）

1. **跨节点题目**：出需要组合多个知识点的综合题
2. **连线提问**：从A知识点推导到B知识点
3. **模拟考试**：完整模拟试卷检验整体掌握

## 优先级排序算法

期末复习时，按以下公式确定复习优先级：

```
优先级 = 影响面 × 考频 × (1 - 掌握度)

影响面：该知识点关联的其他知识点数量（1-5）
考频：历年考试出现的频率（1-5）
掌握度：学生的当前掌握程度（0-1）
```

高优先级 = 影响大 + 常考 + 还没掌握

## 诊断方法

### 快速摸底（5-10分钟）

对每个核心章节问1-2个关键问题：

1. **概念题**：用一句话解释XX
2. **判断题**：XX说法对不对，为什么
3. **应用题**：这个情况应该用什么方法

根据回答快速判断掌握程度：
- 流利回答 → 🟢 已掌握，快速过
- 能答但不自信 → 🟡 需要巩固
- 答错或完全不知道 → 🔴 需要从头学

### 深度诊断

对标记为🟡和🔴的知识点，进一步诊断：

- 是概念不理解？→ 用费曼法重新理解
- 是不知道怎么用？→ 用互动练习巩固应用
- 是记不住？→ 用间隔重复记忆
- 是和其他概念混淆？→ 用对比表格辨析

## 考前冲刺模式

距离考试不足一周时，切换到冲刺模式：

1. **只攻高频考点**：按考频排序，优先复习历年必考
2. **重练习轻讲解**：减少理论讲解时间，增加做题时间
3. **错题优先**：回顾之前的错题，确保不再犯
4. **速记清单**：整理最核心的公式/概念速查表
5. **模拟限时**：做1-2套模拟卷，训练时间分配

## 各学科图谱特点

### 数学：层级递进型
知识点有严格的前后依赖关系，必须按顺序掌握。

### 计算机：组件连接型
知识点相对独立但互相配合，重点是理解组件间的接口和交互。

### 英语：技能网络型
听/说/读/写/词汇/语法相互支撑，需要综合训练。

### 物理：模型关联型
不同章节可能使用相同的物理模型（如守恒定律），重点在模型迁移。

### 化学：微观-宏观映射型
微观结构决定宏观性质，图谱要体现这种对应关系。

### 文科：因果网络型
知识点之间有复杂的因果关系，图谱要展示因果链和多因素交互。
