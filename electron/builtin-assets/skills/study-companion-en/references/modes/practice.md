# 互动练习详细指南

## 核心理念

**做中学（Learning by Doing）。** 知识的理解需要通过实践来检验和巩固。没有经过练习的知识只是"以为自己懂了"——做题是检验理解的试金石。

## 练习设计原则

### 1. 分层递进

练习题按难度分层，确保学生在每个层次都扎实后再进入下一层：

```
⭐ 基础题 → 直接应用核心概念/公式
⭐⭐ 应用题 → 需要识别用哪个知识点，需要1-2步推理
⭐⭐⭐ 综合题 → 需要组合多个知识点，需要多步推理
⭐⭐⭐⭐ 挑战题 → 非标准场景，需要创造性思维
```

### 2. 针对性

- 基于学生的薄弱环节出题，而非随机出题
- 基于用户提供的资料出题，而非脱离资料的通用题
- 优先出高频考点和易错点相关的题目

### 3. 即时反馈

- 答对：追问"为什么"，确认不是蒙的
- 答错：分析错因，针对性讲解，然后给同类题再练
- 半对：肯定对的部分，指出遗漏，引导补全

### 4. 变式训练

同一知识点从不同角度出题，防止"会做题但不会知识"：

- 正向：已知条件求结果
- 逆向：已知结果推条件
- 变形：改变表述方式
- 综合：与其他知识点结合

## 出题流程

### 步骤1：确定出题目标
- 哪个知识点？
- 什么难度层次？
- 考查什么能力？（记忆/理解/应用/分析）

### 步骤2：编写题目
格式：
```
📝 练习题 [⭐难度]

[题目]

A) [选项A]
B) [选项B]
C) [选项C]
D) [选项D]

（或填空题/简答题/计算题格式）

💡 提示（需要时提供）：[渐进式提示]
```

### 步骤3：等待学生作答
给学生思考时间，不要急于提供提示。

### 步骤4：反馈与讲解
```
✅ 正确！/ ❌ 不完全正确

📖 解析：
[详细解释为什么这个答案是对的/错的]

🔑 关键点：
[这道题考查的核心知识点]

💡 避坑指南：
[容易犯的错误和如何避免]

→ 下一题：[同类变式题或进阶题]
```

## 题型设计

### 选择题
适用：概念辨析、快速诊断

设计要点：
- 干扰项要"有道理"——基于常见错误设计
- 避免明显的排除项
- "以上都对/都不对"慎用

### 填空题
适用：关键概念、公式记忆

设计要点：
- 空白处是核心知识点
- 避免填无关紧要的词

### 计算题
适用：数学、物理、化学

设计要点：
- 数据合理，计算量适中
- 重点考查方法而非计算技巧
- 包含单位检验

### 简答题
适用：概念解释、原理论述

设计要点：
- 明确答题要求（字数、要点数）
- 参考答案有评分标准

### 分析题
适用：综合应用、跨章节

设计要点：
- 场景真实，数据合理
- 需要多步骤推理
- 允许合理假设

## 互动节奏控制

### 节奏原则
- 不要连续出超过3道同类型题
- 穿插不同题型和难度
- 每3-5道题做一次小结
- 感觉学生疲劳时，切换学习模式

### 难度调整
- 连续答对3题 → 提升难度
- 连续答错2题 → 降低难度或回到概念讲解
- 答对但犹豫 → 维持当前难度，加追问

### 时间管理
- 基础题：1-2分钟
- 应用题：3-5分钟
- 综合题：5-10分钟
- 如果超时太久，给出提示或拆解问题

## 错题处理

### 错因分类

| 错因类型 | 特征 | 处理方法 |
|---------|------|---------|
| 概念不清 | 答案方向就错了 | 回到概念讲解，用费曼法重新理解 |
| 思路错误 | 方向对但方法错 | 分析正确思路和错误思路的差异 |
| 计算失误 | 方法对但算错了 | 不必过多讲解概念，给同类题再练 |
| 审题不清 | 没看清题目条件 | 训练审题技巧，标注关键词 |
| 知识遗漏 | 不知道某个前置知识 | 补充前置知识，再回来做题 |

### 错题巩固

1. 分析错因
2. 针对性讲解
3. 给一道同类题立即再练
4. 在后续练习中穿插同类题

## 期末复习专项练习

### 真题模拟
- 基于历年真题的题型和难度出题
- 模拟真实考试的时间限制
- 训练时间分配策略

### 薄弱环节突破
- 根据诊断结果集中练习薄弱章节
- 同一知识点多种题型反复练
- 直到正确率稳定在80%以上

### 高频考点速练
- 快速过一遍高频考点的基础题
- 确保基础分不丢
- 时间留给难题和综合题
