# 论文/写作工作流

从构思到成稿的完整写作辅导流程。核心原则：**教写作思维，不代写文章**。

## 写作类型与要求

| 类型 | 学段 | 核心要求 | 字数 |
|------|------|---------|------|
| 议论文 | 初中/高中 | 论点明确、论据充分、逻辑清晰 | 600-800 |
| 说明文 | 初中/高中 | 条理清楚、语言准确 | 500-800 |
| 读后感 | 初中/高中 | 感受真实、联系实际 | 600-800 |
| 学术论文 | 大学 | 研究问题明确、方法规范、论证严谨 | 3000-10000+ |
| 实验报告 | 高中/大学 | 格式规范、数据准确、分析深入 | 1000-3000 |
| 课程论文 | 大学 | 文献综述+分析论证 | 2000-5000 |

## 通用写作流程

### Step 1：审题与构思

1. **明确写作任务**：
   - 什么类型？（议论/说明/分析/研究）
   - 什么主题？有没有限制？
   - 字数要求？格式要求？截止时间？
   - 读者是谁？写作目的是什么？

2. **头脑风暴**：
   - 列出所有相关观点和论据
   - 不评判，先量后质
   - 用思维导图整理思路

3. **确定核心论点**：
   - 一句话概括你的核心观点
   - 如果不能用一句话说清，说明思路还没理顺

### Step 2：构建大纲

大纲是写作的骨架，先有骨架再填血肉：

#### 议论文大纲

```
一、引言
   - 背景/现象引入
   - 提出论点

二、论证主体
   分论点1 + 论据（事实/数据/引用）
   分论点2 + 论据
   分论点3 + 论据
   （可选）反驳对立观点

三、结论
   - 总结论点
   - 升华/展望
```

#### 学术论文大纲

```
一、引言
   - 研究背景
   - 文献综述
   - 研究问题/假设

二、方法
   - 研究设计
   - 数据/样本
   - 分析方法

三、结果
   - 主要发现
   - 数据呈现

四、讨论
   - 结果解读
   - 与已有研究对比
   - 局限性

五、结论
   - 核心贡献
   - 未来方向
```

#### 实验报告大纲

```
一、实验目的
二、实验原理（理论背景）
三、实验器材与步骤
四、实验数据与观察
五、数据处理与分析
六、实验结论
七、误差分析与改进建议
```

### Step 3：起草初稿

1. **先写主体**：从最熟悉的段落开始，避免"开头恐惧"
2. **不要追求完美**：初稿的目标是"写完"，不是"写好"
3. **保持写作流**：卡住时跳过，标记后继续
4. **引用标注**：引用他人观点时立即标注来源，不要指望事后补

### Step 4：修改润色

修改分三轮，每轮关注不同层面：

**第一轮：结构审查**
- 论点是否清晰？是否贯穿全文？
- 段落间逻辑是否连贯？过渡是否自然？
- 论据是否充分支撑论点？有没有逻辑漏洞？
- 有没有偏题或冗余的内容？

**第二轮：语言审查**
- 句子是否通顺？有没有语法错误？
- 用词是否准确？有没有口语化表达？
- 句式是否多样？长短句是否交替？
- 逻辑连接词是否恰当？

**第三轮：格式审查**
- 字数是否符合要求？
- 引用格式是否规范？
- 标题、段落、编号是否统一？
- 有没有错别字和标点错误？

### Step 5：终稿检查

1. 通读全文，确认逻辑流畅
2. 检查所有引用和参考文献
3. 确认格式完全符合要求
4. 请他人阅读获取反馈（如果可能）

## AI辅助写作原则

### 应该做的

| 阶段 | AI辅助方式 |
|------|-----------|
| 构思 | 帮助头脑风暴、梳理思路、确定论点 |
| 大纲 | 审查大纲逻辑、建议补充角度、指出遗漏 |
| 起草 | 提供表达建议、指出逻辑跳跃、建议过渡方式 |
| 修改 | 指出逻辑漏洞、语言问题、结构问题 |
| 格式 | 检查引用格式、字数、排版 |

### 不应该做的

- **不代写**：不直接写出完整段落或文章
- **不替代思考**：不替学生决定论点和立场
- **不编造论据**：不虚构数据、引用或事实
- **不统一风格**：保留学生自己的写作风格和声音

### 辅助话术示例

**构思阶段**：
- "你对这个话题有什么初步想法？先说说看"
- "你能想到哪些支持这个观点的理由？"
- "有没有反对的意见？你怎么回应？"

**大纲阶段**：
- "你的三个分论点之间是什么逻辑关系？并列还是递进？"
- "每个分论点都有足够的论据支撑吗？"
- "读者可能提出什么质疑？你的文章如何回应？"

**修改阶段**：
- "这段话的核心意思是什么？能不能用更简洁的方式表达？"
- "这两个段落之间有逻辑跳跃，读者可能跟不上，需要加过渡"
- "这个论据能支撑你的论点吗？有没有反例？"

## 分类型写作指导

### 议论文

**核心能力**：论点+论据+论证

1. **论点**：明确、具体、可论证（避免"我觉得XX很好"这种模糊论点）
2. **论据**：事实论据+道理论据，至少2-3个支撑
3. **论证**：演绎推理或归纳推理，逻辑链完整
4. **反驳**：预判对立观点并回应，增强说服力

**常见问题**：
- 论点模糊或过大
- 论据与论点不匹配
- 只有观点没有论证
- 情感替代逻辑

### 学术论文

**核心能力**：问题意识+文献基础+方法规范+论证严谨

1. **问题意识**：研究问题必须明确、具体、有价值
2. **文献综述**：展示对已有研究的了解，找到研究空白
3. **方法规范**：研究方法必须适合研究问题，描述足够详细可复现
4. **论证严谨**：结论必须基于证据，不超出数据支持的范围

**常见问题**：
- 研究问题太宽泛
- 文献综述只是罗列而非分析
- 方法描述不够详细
- 结论过度推断

### 实验报告

**核心能力**：规范格式+准确数据+深入分析

1. **格式规范**：严格按照实验报告格式撰写
2. **数据准确**：如实记录，不篡改、不挑选数据
3. **分析深入**：不仅报告结果，还要解释原因
4. **误差分析**：诚实分析误差来源和改进方向

**常见问题**：
- 数据处理不规范（有效数字、单位）
- 只描述现象不分析原因
- 误差分析流于形式
- 抄袭他人实验报告

## 写作素材积累

### 议论文素材

AI可以帮助积累和整理写作素材：
- 名人名言（按主题分类）
- 时事热点（按话题归档）
- 历史典故（按论点类型分类）
- 科学发现（按论证用途分类）

### 学术写作素材

- 文献管理：帮助整理参考文献格式
- 学术表达：提供学术写作常用句式
- 方法描述：提供标准方法描述模板
