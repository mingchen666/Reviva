# 错题分析工作流

从错题中最大化学习效果的流程。核心理念：**做错题不是失败，分析错题才是最有效的学习**。

## 工作流步骤

### Step 1：记录错题

每道错题记录以下信息：

```
📌 错题记录

题目：[原题]
我的答案：[我写的答案]
正确答案：[标准答案]
科目/章节：[所属知识点]
```

### Step 2：错因分类

不要简单说"算错了"，而是精确定位错误类型：

| 错因类型 | 特征 | 发生率 | 补救策略 |
|---------|------|--------|---------|
| **概念错误** | 方法方向就错了，用了错误的公式/定理 | 30% | 回到概念重新理解，用费曼法检验 |
| **审题错误** | 漏看条件、误解题意、看错数字 | 25% | 训练审题技巧：标记关键字、复述题意 |
| **计算失误** | 方法对但算错了 | 20% | 不需要重新学概念，多练计算 |
| **逻辑跳跃** | 跳过必要步骤，推导不完整 | 15% | 训练写完整解题过程 |
| **知识遗漏** | 不知道某个前置知识点 | 10% | 补充缺失的前置知识 |

### Step 3：根因分析

对每道错题追问"为什么"至少3层：

```
示例：
错题：求积分 ∫x·e^x dx，我用了分部积分但答案不对

第1层为什么：我选错了u和v
第2层为什么：我不确定怎么选u和v
第3层为什么：我没掌握LIATE规则（选u的优先级）

根因：分部积分的u/v选择规则没掌握
补救：学习LIATE规则，做5道分部积分专项练习
```

### Step 4：针对性补救

根据错因类型选择补救方法：

| 错因 | 补救方法 | 工具 |
|------|---------|------|
| 概念错误 | 费曼法重新理解→做2道同类题 | 苏格拉底追问 |
| 审题错误 | 练习"审题30秒"法：读题后先复述再动手 | 出审题陷阱题 |
| 计算失误 | 做3道同类计算题，要求每步检验 | 限时计算训练 |
| 逻辑跳跃 | 写完整解题过程，每步写依据 | 对照标准过程 |
| 知识遗漏 | 补充前置知识，再回来做原题 | 概念学习工作流 |

### Step 5：同类题巩固

1. 找到与错题相同考点和题型的2-3道题
2. 用正确的解题方法做这些题
3. 如果连续2道做对，说明补救成功
4. 如果仍然做错，回到Step 3重新分析

### Step 6：间隔复习

1. 将错题制成闪卡（正面：题目+错因，背面：正确思路+关键步骤）
2. 按间隔重复节奏复习：
   - 补救后1天：重做原题
   - 3天后：做一道同类变式题
   - 1周后：再做一道变式题
3. 生成HTML错题本，按错因分类浏览

## 错题本生成

AI可以为用户生成结构化的HTML错题本：

```html
错题本功能：
- 按科目/章节/错因分类浏览
- 每题显示：原题、我的答案、正确答案、错因分析、补救记录
- "重做"按钮：隐藏答案重新做题
- 统计面板：各错因类型的占比图表
- 导出功能：打印/导出PDF
```

## 错题分析的常见误区

1. **只看答案不看过程**：答案对了不代表过程对，答案错了更要看过程
2. **归因为"粗心"**：说"粗心"是逃避分析，要具体到是审题错、计算错还是跳步错
3. **只改答案不找原因**：把错误答案改成正确答案 ≠ 理解了为什么错
4. **错题只看一遍**：不复习的错题本等于没做，要间隔重复
5. **同类错题重复出现**：说明根因分析不够深，或者补救练习不够
