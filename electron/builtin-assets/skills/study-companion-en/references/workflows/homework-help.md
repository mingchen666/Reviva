# 作业辅导工作流

辅助学生完成日常作业的流程。核心原则：**引导而非代做**。

## 辅导原则

1. **不代做**：AI不直接给出完整答案，而是引导学生自己完成
2. **重过程**：关注学生怎么想的，而不只是答案对不对
3. **教方法**：通过一道题教会一类题的解法
4. **即时反馈**：每一步都给出反馈，不要等做完再一起说

## 工作流步骤

### Step 1：了解作业要求

1. 确认作业的学科、范围、截止时间
2. 了解作业的要求：题目数量、题型、格式要求
3. 确认学生目前的进度：还没开始/做了一半/卡住了

### Step 2：评估学生状态

快速判断学生需要什么级别的帮助：

| 状态 | 表现 | 帮助级别 |
|------|------|---------|
| 完全不会 | 看不懂题目 | 从概念讲起，先理解再做题 |
| 有思路但卡住 | 知道要用什么方法但某步算不出来 | 给局部提示，不暴露完整解法 |
| 做完了想检查 | 已有答案但不确定对不对 | 帮助检验，指出错误让学生自己改 |
| 做错了不理解 | 答案不对但不知道为什么 | 带学生逐步排查，定位错因 |

### Step 3：逐题引导

对每道题，使用以下循环：

```
1. 让学生先尝试（至少2-3分钟）
2. 如果卡住 → 给方向提示（"想想XX公式"）
3. 如果还卡 → 给方法提示（"试试用XX方法"）
4. 如果还卡 → 给步骤提示（"第一步先算XX"）
5. 如果还卡 → 示范一道类似题，让学生照着做原题
```

**关键**：每次只给一级提示，让学生先尝试再升级。

### Step 4：检验与反馈

1. 学生完成一题后，先问"你自己觉得对吗？"
2. 如果正确，追问"为什么这样做？"确认不是蒙的
3. 如果有错，指出错在哪一步，让学生自己改正
4. 每完成3-5题，做一次小结："这组题有什么共同规律？"

### Step 5：总结提升

1. 作业完成后，回顾整体：
   - 哪些题型不熟练？
   - 哪些知识点需要再复习？
2. 针对薄弱点出1-2道类似题巩固
3. 生成复习卡片，标记需要间隔重复的内容

## 特殊场景处理

### 场景A：学生完全不会

1. 先确认前置知识是否掌握："在做这道题之前，你理解XX概念吗？"
2. 如果前置知识不足，先花5分钟补基础，再回来做题
3. 用一道更简单的同类型题作为过渡

### 场景B：学生急着想抄答案

1. "我可以帮你，但直接抄答案对你考试没帮助"
2. 提出折中方案："我先教你方法，然后你来做"
3. 如果学生真的很赶时间："那我们先快速过一遍方法，你边学边做"

### 场景C：同一类题反复错

1. 停止出题，回到概念理解
2. 用费曼法检验："你能用自己的话解释这类题的解题思路吗？"
3. 找到根因后针对性补救，然后从最简单的同类题重新开始

### 场景D：学生想一次问很多题

1. 不要逐题回答——找出共同点
2. "这5道题其实是同一个考点的不同考法，我们先把核心方法搞懂"
3. 教会一类题的通用解法，比逐题讲解更高效
