# 化学学习策略

## 学科特点

化学是"从微观粒子理解宏观性质"的学科。兼具理科的逻辑性和一定的记忆量。核心能力是微观-宏观的对应思维。

## 学习层次

```
L0 记忆 → 记住化学符号、反应方程式
L1 理解 → 理解反应原理和物质结构
L2 应用 → 能预测反应结果和物质性质
L3 分析 → 能分析复杂反应体系
L4 设计 → 能设计合成路线和实验方案
```

## 核心学习策略

### 微观-宏观对应

1. **宏观现象**：观察/描述实验现象
2. **微观机制**：用原子/分子/电子层面解释
3. **符号表达**：用化学方程式/结构式精确表示
4. **三者对应**：建立宏观现象↔微观机制↔符号表达的完整链路

### 反应学习：分类 → 机制 → 条件 → 应用

1. **分类归属**：属于哪类反应（氧化还原/酸碱/配位等）
2. **理解机制**：电子转移/质子转移/配位键形成等
3. **掌握条件**：什么条件下发生，为什么需要这些条件
4. **应用场景**：在工业/实验室/生活中的应用

### 记忆策略：理解 > 硬背

- 元素周期律：理解规律后推理，而非逐个记忆
- 化学方程式：理解反应类型和机制后自然记住
- 物质性质：基于结构推导性质，建立因果链

## 分科目策略

### 无机化学
- **核心直觉**：元素周期律是"导航地图"，结构决定性质
- **可视化重点**：周期表分区图、晶体结构图、轨道形状图
- **常见误区**：死背性质不理解结构原因；混淆配位键和共价键
- **复习重点**：原子结构、分子结构、元素周期律、酸碱理论、配位化学

### 有机化学
- **核心直觉**：官能团决定性质，反应是电子的重新分配
- **可视化重点**：反应机理箭头图、构象分析图、合成路线图
- **常见误区**：只记反应不理解机理；混淆亲核和亲电
- **复习重点**：命名、异构、反应机理、官能团转化、合成设计

### 物理化学
- **核心直觉**：热力学告诉你"能不能"，动力学告诉你"快不快"
- **可视化重点**：相图、能量分布图、反应坐标图
- **常见误区**：混淆ΔG和ΔG°；不理解活化能的物理意义
- **复习重点**：热力学、相平衡、化学平衡、电化学、动力学

### 分析化学
- **核心直觉**：分析是"问物质是什么和有多少"
- **可视化重点**：滴定曲线、光谱图、色谱图
- **常见误区**：混淆准确度和精密度；忽略系统误差
- **复习重点**：误差分析、滴定分析、光谱分析、色谱分析

## 可视化讲解应用

化学的微观世界需要大量可视化：

- **分子结构**：用ASCII展示原子连接方式
- **反应机理**：用箭头图展示电子流动
- **周期表分区**：用颜色分区展示元素性质递变

示例 — 甲烷分子结构：
```
       H
       |
   H - C - H    四面体结构
       |
       H
键角：109.5°
```

## 间隔重复记忆应用

化学中大量需要记忆的内容适合间隔重复：

- 元素符号和原子序数
- 常见离子和化合价
- 关键反应方程式
- 官能团特征反应
- 重要常数和公式

## 互动练习出题指南

### 基础题（⭐）
- 直接应用化学方程式和概念
- 例：写出Na与水反应的化学方程式
- 推荐题型：fill, choice
- 认知层级：L1-L2

### 应用题（⭐⭐）
- 需要判断反应类型并选择方法
- 例：判断某反应是否为氧化还原反应，标出电子转移方向
- 推荐题型：choice, match
- 认知层级：L2-L3

### 综合题（⭐⭐⭐）
- 多步反应推断、实验方案评价
- 例：根据实验现象推断未知物质
- 推荐题型：choice, order（排列实验步骤）
- 认知层级：L3-L4

### 挑战题（⭐⭐⭐⭐）
- 合成路线设计、实验方案优化
- 例：设计从乙烯到乙酸乙酯的最优合成路线
- 推荐题型：order（排列合成步骤）, fill
- 认知层级：L4-L5

### 化学特有题型

- **方程式配平题**（order类型）：排列配平步骤
- **物质推断题**（match类型）：现象与物质配对
- **实验评价题**（choice/tf）：判断实验操作是否正确
- **微观-宏观对应题**（match类型）：宏观现象与微观机制配对

## HTML可视化工具

### 3Dmol.js分子结构3D查看
WebGL分子查看器，支持PDB/SDF/MOL2格式：
```html
<script src="https://cdn.jsdelivr.net/npm/3dmol@2/build/3Dmol-min.js"></script>
<div id="viewer" style="width:400px;height:400px;"></div>
<script>
  let viewer = $3Dmol.createViewer(document.getElementById('viewer'));
  viewer.addModel(data, "sdf"); // 或pdb格式
  viewer.setStyle({},{stick:{}});
  viewer.zoomTo();
  viewer.render();
</script>
```

### 交互式周期表
CSS Grid布局的HTML周期表：
- 点击元素展开详情面板（原子序数、电子构型、电负性、常见化合价）
- 颜色编码：按族/按区块(s/p/d/f)/按金属非金属
- 筛选搜索功能

### SVG反应机理图
用SVG绘制箭头推动机理（curly arrows），标注电子流动方向

### 化学方程式配平器
JS实现矩阵法化学方程式配平，输入未配平方程→输出配平结果+步骤
