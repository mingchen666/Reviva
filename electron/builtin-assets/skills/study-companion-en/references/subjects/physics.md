# 物理学习策略

## 学科特点

物理是"用数学语言描述自然规律"的学科。核心能力是物理直觉+数学工具+实验思维的三位一体。

## 学习层次

```
L0 现象 → 知道物理现象是什么
L1 概念 → 理解物理概念的含义
L2 规律 → 掌握物理定律和公式
L3 应用 → 能用定律分析具体问题
L4 综合 → 能组合多个定律解决复杂问题
L5 建模 → 能对实际问题建立物理模型
```

## 核心学习策略

### 物理直觉优先

1. **从现象出发**：先观察/想象物理现象
2. **建立直觉**：用生活经验和常识理解现象
3. **精确描述**：用物理概念和定律精确描述
4. **数学表达**：用数学公式定量表达
5. **验证预测**：用公式预测结果，与直觉对照

### 解题方法：模型 → 方程 → 求解 → 检验

1. **建立模型**：明确研究对象、受力分析、状态分析
2. **列方程**：根据物理定律列方程（牛顿定律、守恒定律等）
3. **数学求解**：解方程得到定量结果
4. **物理检验**：检查量纲、数量级、极端情况是否合理

### 可视化核心

物理几乎每个知识点都需要可视化：
- **力学**：受力分析图、运动轨迹图、能量转化图
- **电磁学**：电场线图、磁力线图、电路图
- **光学**：光路图、干涉衍射图
- **热学**：PV图、热力学循环图

## 分科目策略

### 力学
- **核心直觉**：力是改变运动状态的原因，能量是做功的本领
- **可视化重点**：受力分析图（FBD）、v-t图、能量守恒图
- **常见误区**：混淆速度和加速度方向；忘记考虑所有受力
- **复习重点**：牛顿三定律、动量守恒、能量守恒、刚体力学

### 电磁学
- **核心直觉**：电荷产生场，场对电荷施力
- **可视化重点**：电场线/磁力线分布、电路拓扑图
- **常见误区**：混淆电势和电势能；不理解感应电动势的方向
- **复习重点**：库仑定律、高斯定理、安培定律、法拉第定律、麦克斯韦方程组

### 热学与统计物理
- **核心直觉**：宏观性质是微观统计平均的结果
- **可视化重点**：PV图上的过程曲线、麦克斯韦速度分布
- **常见误区**：混淆温度和热量；不理解熵的统计意义
- **复习重点**：热力学定律、理想气体、卡诺循环、统计分布

### 量子力学
- **核心直觉**：微观世界由概率描述，不确定性是内在的
- **可视化重点**：波函数图像、势阱中的概率分布
- **常见误区**：把波函数理解为物理波；不理解测量对状态的影响
- **复习重点**：薛定谔方程、算符、氢原子、自旋

## 苏格拉底提问应用

物理特别适合用苏格拉底法引导理解：

**示例：理解牛顿第三定律**
- "当你推墙的时候，墙也在推你吗？"
- "那为什么你往后退而墙不动？"
- "如果力和反作用力大小相等方向相反，它们会抵消吗？"
- "它们作用在同一个物体上吗？"

通过层层追问，让学生自己发现"作用力和反作用力作用在不同物体上"这个关键点。

## 互动练习出题指南

### 基础题（⭐）
- 直接应用物理公式计算
- 例：已知F=10N, m=2kg，求加速度a=？
- 推荐题型：choice, fill
- 认知层级：L2（理解）

### 应用题（⭐⭐）
- 需要选择正确的物理模型和公式
- 例：斜面上物体受力分析，求加速度
- 推荐题型：choice, fill
- 认知层级：L3（应用）
- 关键：学生需要先建模（确定坐标系、受力分解）再计算

### 综合题（⭐⭐⭐）
- 多过程问题，需要分段列方程
- 例：抛体运动+碰撞+弹簧，分段分析
- 推荐题型：choice, order（排列分析步骤）
- 认知层级：L4（分析）
- 关键：考查学生能否将复杂问题分解为子过程

### 挑战题（⭐⭐⭐⭐）
- 非标准模型，需要建立物理模型
- 例：电磁感应中的变加速运动，需用能量方法而非运动学
- 推荐题型：choice, fill
- 认知层级：L5（综合）
- 关键：标准方法不适用，需要创造性地选择方法

### 物理特有题型

- **量纲检查题**：判断表达式量纲是否正确（choice/tf）
- **估算题**：对物理量的数量级进行估算（choice）
- **模型识别题**：给定情境判断适用哪个物理模型（choice/match）
- **实验评价题**：评价实验方案的合理性（choice/tf）

### 出题注意事项

1. 所有计算题答案必须检验量纲是否正确
2. 数值答案要检查数量级是否合理
3. 干扰项应包含常见计算错误（如漏掉sin/cos、混淆g和2g）
4. 物理情境要贴近生活或实验实际

## HTML可视化工具

### PhET交互式模拟嵌入
PhET提供400+免费HTML5物理/化学/生物模拟，可直接iframe嵌入：
```html
<iframe src="https://phet.colorado.edu/sims/html/circuit-construction-kit-dc/latest/circuit-construction-kit-dc_all.html"
        width="800" height="600"></iframe>
```
常用模拟：电路、力与运动、波、能量、光学

### Canvas物理模拟
可生成基于Canvas的简单物理模拟HTML：
- 抛体运动（requestAnimationFrame + 参数滑块）
- 弹簧振动（胡克定律动画）
- 波的干涉（正弦波叠加可视化）

### SVG力学图
生成SVG受力分析图、光路图、电路图：
- 力矢量：SVG `<line>` + 箭头标记
- 光路：SVG `<path>` + 反射/折射

### Chart.js数据图
实验数据实时绘图：`<script src="https://cdn.jsdelivr.net/npm/chart.js@4"></script>`
适用于PV图、v-t图、能量转换图等
