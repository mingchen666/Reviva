# 英语学习策略

## 学科特点

英语是典型的"输入+输出"双轮驱动学科。单纯背单词或看语法规则是不够的，必须在真实的语言环境中大量接触和主动使用。教学语言以英语为主，中文辅助解释。

## 核心能力模型

```
听力理解 → 能听懂不同语速和口音
口语表达 → 能流利表达想法
阅读理解 → 能理解各类文本
写作能力 → 能写出结构清晰的文本
词汇积累 → 有足够的词汇量支撑以上四项
语法掌握 → 有正确的语法意识
```

## 核心学习策略

### 词汇学习：语境 > 孤立记忆

1. **在句子中学习**：每个单词至少放在一个完整句子中
2. **词族扩展**：学一个词时，同时学习其变形（名词/动词/形容词/副词）
3. **搭配学习**：记住常见搭配（collocation），而非孤立词义
4. **多义辨析**：一个词的多个意思通过不同例句区分
5. **词根词缀**：理解构词法，实现批量记忆

### 语法学习：规则 → 示例 → 应用 → 内化

1. **理解规则**：清晰呈现语法规则，中英对照
2. **大量示例**：用多个真实例句展示用法
3. **主动应用**：造句练习，即时纠正
4. **内化使用**：在自由表达中自然使用，不再刻意想规则

### 阅读学习：整体理解 > 逐字翻译

1. **先抓大意**：快速浏览获取文章主旨
2. **再析细节**：精读关键段落，理解细节
3. **推断词义**：根据上下文推断生词意思
4. **分析结构**：理解文章的逻辑结构和论证方式

### 写作学习：模仿 → 改写 → 创作

1. **分析范文**：理解优秀写作的结构和表达
2. **改写练习**：变换角度或表达方式重写
3. **独立创作**：根据题目要求独立写作
4. **对比修改**：将自己的写作与范文对比，找出改进空间

## 分场景策略

### 四六级备考
- **听力**：精听+泛听结合，重点训练数字、地点、态度题
- **阅读**：训练快速定位和推断能力，掌握题型策略
- **写作**：背诵框架模板，但必须学会灵活变通
- **翻译**：中英对照练习，注意中式英语问题

### 考研英语
- **长难句**：逐句分析，掌握从句嵌套结构
- **阅读理解**：重点训练作者态度和推断题
- **写作**：大作文注重论点展开，小作文注重格式规范

### 日常英语提升
- **沉浸式学习**：用英语讨论感兴趣的话题
- **场景模拟**：模拟真实场景对话（面试、演讲、日常交流）
- **跟读模仿**：跟读英语材料，改善发音和语调

## 双语教学原则

- **讲解用英语**：知识点讲解使用英语，培养学生英语思维
- **辅助用中文**：复杂概念首次出现时用中文辅助解释
- **术语中英对照**：关键术语同时给出英文和中文
- **渐进式纯英语**：随着学习深入，逐步减少中文辅助

示例：
```
📖 Word: "ubiquitous" /juːˈbɪkwɪtəs/

English definition: present, appearing, or found everywhere
中文释义：无处不在的

Example: Smartphones have become ubiquitous in modern society.
（智能手机在现代社会已经无处不在。）

Word family: ubiquity (n.)
Collocation: ubiquitous presence / ubiquitous access
```

## 互动练习出题指南

### 基础题（⭐）
- 词汇辨析、语法规则识别
- 例：选择"accept"和"except"的正确用法
- 推荐题型：choice, fill
- 认知层级：L1-L2

### 应用题（⭐⭐）
- 语法应用、语境中的词汇选择
- 例：在给定段落中选择正确的时态
- 推荐题型：choice, fill（Cloze）
- 认知层级：L2-L3

### 综合题（⭐⭐⭐）
- 阅读理解、语法改错、翻译
- 例：阅读文章后回答推断题
- 推荐题型：choice, fill
- 认知层级：L3-L4

### 挑战题（⭐⭐⭐⭐）
- 写作评价、长难句分析、学术表达
- 例：判断论文摘要中语法和表达的错误
- 推荐题型：fill（关键词论述），tf
- 认知层级：L4-L5

### 英语特有题型

- **Cloze完形填空题**（fill类型）：多空上下文填空
- **阅读理解题**（choice类型）：文章+细节/推断/主旨题
- **语法改错题**（tf/choice类型）：识别并修正语法错误
- **词汇辨析题**（choice类型）：近义词/形近词辨析

### 互动练习类型

### 词汇练习
- 根据英文释义写单词
- 根据语境选择正确词义
- 用给定词组造句
- 词根词缀推导词义

### 语法练习
- 改错题（识别语法错误）
- 填空题（选择正确语法形式）
- 句型转换（主动被动、直接间接引语等）
- 造句练习（使用指定语法结构）

### 阅读练习
- 主旨题
- 细节题
- 推断题
- 词汇题（根据上下文猜词义）

### 写作练习
- 句子改写
- 段落写作
- 缩写/扩写
- 完整作文（附评分标准反馈）

## 间隔重复记忆应用

英语词汇特别适合间隔重复：

1. **新词卡片**：正面英文（+发音），背面中文释义+例句
2. **复习节奏**：
   - 第1次：学完后5分钟
   - 第2次：30分钟后
   - 第3次：12小时后
   - 第4次：1天后
   - 第5次：3天后
3. **记忆技巧**：
   - 联想记忆：将新词与已知词汇/经历关联
   - 故事记忆：将一组词编入一个故事
   - 图像记忆：为抽象词创造心理图像

## 费曼学习法应用

英语学习中费曼法的应用：

- **语法解释**："用英语解释现在完成时和过去时的区别，就像教一个初学者"
- **文章复述**："用自己的话复述刚才读的文章，不用原文的句子"
- **词汇教学**："给一个不懂英语的人解释'however'和'although'的区别"

这些练习既锻炼理解，又锻炼表达能力。

## HTML可视化工具

### HTML闪卡生成
生成带CSS翻转动画和SM-2间隔重复的自包含HTML闪卡应用：
- 卡片数据：`{front: "ubiquitous", back: "无处不在的", tags: "CET-4 vocabulary"}`
- 翻转动画：CSS 3D transform rotateY
- SM-2算法：自动调度复习时间
- localStorage：跨会话保存进度
- 参考 `scripts/flashcard-template.html` 模板

### Anki TSV导出
生成可直接导入Anki的TSV文件，格式：
```
正面[TAB]背面[TAB]标签
ubiquitous[TAB]无处不在的[TAB]CET-4::vocabulary
```
字段中支持HTML富文本（如`<b>`, `<i>`, `<br>`）

### 填空题（Cloze Deletion）
生成HTML填空练习，比选择题更有效（测试回忆而非识别）：
```html
<input type="text" data-answer="were" placeholder="___">
If I ___ you, I would accept the offer.
```

### 语音API（Web Speech API）
浏览器原生语音识别，用于发音练习：
- `SpeechRecognition`：识别学生朗读
- `SpeechSynthesis`：播放标准发音
- 适用于Chrome/Edge，无需安装
